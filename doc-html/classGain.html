Definition of a gain function:

Given a finite, non-empty set X (of possible secret values) and a non-empty set W (of possible actions), a gain function is a function g: W x X → R. The idea is that g(w, x) specifies the gain that the adversary achieves by taking action w when the actual value of the secret is x.

A Gain object has 3 attributes:

  • prior : A pointer to a Distribution object, which represents the prior probability distribution on the set of secrets.
  • matrix : A matrix w x n where w is the number of actions that the adversary can take and n is the number of secrets.
  • num_act : Number of distinct actions that an adversary can take.

Function documentation

Gain::Gain()

Default constructor.

This constructor creates a Gain object with num_act = 0, prior = NULL and empty matrix.

Gain::Gain(Distribution& prior, std::string file)

Construtor used when the gain function matrix is in a file.

Parameters
prior Prior distribution on the set of secrets
file File name which contains a gain function matrix.

The file format must be the following:

w n
g11 g12 ... g1n
g21 g22 ... g2n
...
gw1 gw2 ... gwn

where w is num_act, and n is the number of secrets. Each gain gij is the gain of taking action i when the secret value is j. Each two numbers must be separated by a space (" ").

Warning

The number of columns in the gain function matrix must be the same as prior -> num_el.

Gain::Gain(Distribution& prior, int num_act, int MIN, int MAX)

Constructor used to generate a random gain function.

Parameters
prior Prior distribution on the set of secrets.
num_act Number of actions.
MIN Lower bound for the gain of an action.
MAX Upper bound for the gain of an action.

The parameters MIN and MAX must form an interval. All the gains will be generated randomly in the interval [MIN,MAX].

std::string Gain::toString(int precision)

Generates a string with the gain function matrix.

Parameters
precision Decimal precision for long double numbers. For example, use the value 3 to print 0.322, use 5 to print 0.32197, and so on.
Returns

The gain function matrix is returned in the following format:

g11 g12 ... g1n
g21 g22 ... g2n
...
gw1 gw2 ... gwn

where each gain gij is the gain of taking action i when the secret value is j. Each two numbers are separated by a space (" ").

Variable documentation

int Gain::num_act

Number of distinct actions that an adversary can take.

This attribute is exactly the number of rows in matrix.

Distribution* Gain::prior

A pointer to a probability distribution on a set of secrets.

It is a distribution on the set of secrets. See Distribution for more details.

std::vector<std::vector<long double>> Gain::matrix

Gain function matrix.

A matrix w x n where w is the number of actions that the adversary can take and n is the number of secrets. Each value matrix[i][j] is the gain that the adversary achieves by taking action i when the actual value of the secret is j.

The rows and columns are indexed from the position 0 (1st element) to the position num_act-1 (for rows) or prior ->num_el-1 (for columns).
